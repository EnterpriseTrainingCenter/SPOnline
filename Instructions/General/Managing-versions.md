# Managing document versions

1. In a document library, either select a file and click **Version history** on the tool bar (you might have to click the ellipsis to see the command) or right-click a file or click the ellipsis beside the document and select **Version history** from the context menu.
1. Now you can manage the versions.

    * View a version: Click on the date of the version you want to view.
    * Restore a version: Click the drop-down beside the date of the version you want to restore and select **Restore** from the context menu. Confirm by clicking OK.
    * Delete a version: Click the drop-down beside the date of the version you want to delete and select **Delete** from the context menu. Confirm by clicking OK.

1. Close the version history panel by clicking the X in the upper right corner.
