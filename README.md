# Implementing and administering SharePoint Online and OneDrive for Business Lab Guide

This repository contains practices and labs. Practices are intended to be demonstrated by the instructor and followed by the course participants. Labs are intended to be performed by the course participants alone.

The PowerPoint presentation for this course contains direct links to the practices and labs. Please use these links, because your instructor may change the order of modules, practices and labs.

See also Issues, not only containing current known issues with possible solutions, but also allows to raise new issues.

If you are a trainer or participant of the course, you may fork this repository. Pull requests are accepted after review. Please follow our Guidelines.
